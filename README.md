# 🛒 Sistema de Ventas
Sistema de gestión de ventas desarrollado con Python, SQL Server y Streamlit.

## 🖥️ Tecnologías
- Python 3.14
- SQL Server
- Streamlit
- Pandas
- Pyodbc

## 📋 Módulos
- 👥 Clientes — registro y gestión
- 📦 Inventario — productos y stock
- 🧾 Ventas — carrito, historial y resumen

## ▶️ Cómo ejecutar
1. Instalar dependencias: `pip install streamlit pyodbc pandas`
2. Configurar conexión en `database/connection.py`
3. Ejecutar: `python -m streamlit run app.py`
